# 类型介绍

您可以通过 config.js 外观描述中定义的 type 字段，来定义控件的类型和配置。

# 类型概览

type 字段支持以下类型：

类别类型说明
基础类型number数值输入框，可以定制步长、最大值和最小值、后缀。
input文本输入框，支持单行、多行模式。
select选择器，支持自定义下拉选项。
slider滑动条，可以定制步长、最大值和最小值、后缀。
switch开关，支持是否展示文字状态。
position可以控制元素的水平或垂直位置。
color颜色选择器，支持主题定义。
bgcolor颜色选择器，支持透明、纯色、渐变色、主题定义。
radioGroup一组单选框
checkboxGroup一组复选框
checkbox复选框，支持是否展示文字状态。
image图片上传
media媒体资源上传
unit单位，支持自定义单位项。
icon图标选择器
code自定义内容，返回一个 render 函数。
组合类型radius圆角，由四个输入框组成，可定制左上、右上、右下、左下四个圆角值。
margin外边框，由四个输入框组成，可定制上右下左四个外边距值。
padding内边框，由四个输入框组成，可定制上右下左四个内边距值。
size定义容器尺寸
textStyle文本样式设置，可以设置字体大小、颜色、粗细、是否斜体、对齐方式。
自定义类型slot预留插槽

# number

数值输入框，可以定制步长、最大值和最小值、后缀。

配置项说明：
字段名含义类型是否必须默认值备注
name名称stringfalse
type类型stringtrue
max最大值numberfalse
min最小值numberfalse
step步长numberfalse
unit后缀stringfalse
配置示例：
{
  "name": "数字框",
  "type": "number",
  "unit": "px"
}
值说明：

# color

颜色选择器，支持主题定义。

配置项说明：
字段名含义类型是否必须默认值备注
name名称stringfalse
type类型stringtrue
theme主题配置objectfalse根据 theme 配置，初始化颜色默认值。
theme对象属性
key主题字段stringtrue主题字段，具体参数主题模块帮助。
opacity透明度numberfalse1颜色透明度，0-1的值。
配置示例：
{
  "name": "颜色选择",
  "type": "color",
  "theme": {
    "key": "color",
    "opacity": 1
  }
}
值说明：

# bgcolor

颜色选择器，支持透明、纯色、渐变色、主题定义。

配置项说明：
字段名含义类型是否必须默认值备注
name名称stringfalse
type类型stringtrue
theme主题配置objectfalse根据 theme 配置，初始化颜色默认值。
theme对象属性
type类型stringfalselinearlinear：线性渐变，radial：径向渐变
degree旋转角度numberfalse0旋转角度
points一组点集合Arrayfalse每一项都包含key和opacity，key表示主题字段，opacity表示透明度。
配置示例：
{
  "name": "颜色选择",
  "type": "bgcolor",
  "theme": {
    "type": "linear",
    "degree": 90,
    "points": [
      {
        "key": "color",
        "opacity": 0.3
      },
      {
        "key": "color",
        "opacity": 0.1
      }
    ]
  }
}
值说明：
{
  "style": "linear-gradient(90deg,rgba(17,95,234,0.3) 0%,rgba(17,95,234,0.1) 100%"
}

# checkboxGroup

一组复选框。

配置项说明：
字段名含义类型是否必须默认值备注
name名称stringfalse
type类型stringtrue
options选项列表Arraytrue每一项都包含text、value和key。text表示显示文本，value表示文本的值，key表示字段名称。
配置示例：
{
  "name": "复选框",
  "type": "checkboxGroup",
  "options": [
    {
      "value": 1,
      "text": "选项一",
      "key": "field1"
    },
    {
      "value": 2,
      "text": "选项二",
      "key": "field2"
    }
  ]
}
值说明：
{
  "field1": true,
  "field2": true
}
